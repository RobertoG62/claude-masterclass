### Slide 2: What is Claude Code?

- Official CLI tool from Anthropic
- AI assistant that lives in your terminal
- Direct access to your codebase and tools
- Natural language interface for development tasks

**Content:**
- Read, write, and edit files
- Execute terminal commands
- Search and navigate codebases
- Git operations and version control
- Web fetching and research

- API key configuration
- First-time setup wizard
- IDE integrations available

- Glob and Grep for searching
- Task for complex operations
- WebFetch and WebSearch for research

### Slide 7: Effective Prompting
**Content:**
- Be specific about what you want

- Reference specific files when needed

- Use slash commands for common operations

### Slide 8: Code Reading & Navigation
**Content:**
- Use Read tool to examine files
