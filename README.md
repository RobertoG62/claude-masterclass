# Claude Code Masterclass

Comprehensive guide and documentation for mastering Claude Code - the AI-powered CLI development assistant.

## What's Inside

| File | Description |
|------|-------------|
| `CLAUDE.md` | Project intelligence rules for high productivity |
| `presentation_plan.md` | 16-slide presentation structure for teaching Claude Code |
| `.claude/skills/refactor-protocol.md` | Step-by-step guide for safe code refactoring |

## Quick Start

1. Clone this repository
2. Copy `CLAUDE.md` to your project root to establish productivity rules
3. Use the skills in `.claude/skills/` as templates for your own workflows
